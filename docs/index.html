Description

This plugin allows you to replace the standard selected on their own, which can be styled as your heart desires. Default stylized standard select. Mechanic behavior as much as possible trying to match the standard select'у.

Initialization

new Select(el, options);

Demo

Sort:

Options

animations : ''

Animates the appearance of the list type:string. Can have the following options:

  • slideInUp
  • zoomInDown
  • slideInDown
  • fadeIn

Default fulfills without animation. If desired / required, you can expand the number of its animations.

maxWidth :

It allows you to set the maximum width of the drop-down list type:integer. If you specify a maximum width of less than or equal to the original selecta, the maximum width of the original selecta will be exhibited.

onChange : function(value, text, index){}

This method allows you to get a value of the selected item.

Demos

Change callback

new Select(el, {
  onChange: function(event, value, text, index) {
    alert('Event: ' + event.type + '\nYou choised (' + text + ')\nIndex(' + index + ')\nValue(' + value + ')')
  }
});

Show max items

new Select(el, {
  showMaxOptions: 3
});

Max width

new Select(el, {
  maxWidth: 350
});

Destroy

var select = new Select(el, {
  maxWidth: 350
});
// some time later
select.destroy();

New option

var select = new Select(el, {
  maxWidth: 350
});
// Adding new option into system select
el.innerHTML += '<option value="new_option">New option</option>';
// update custom select
select.update();

Custom styles

var select = new Select(el, {
  customClass: 'custom' // for example
});

Transitions

var select = new Select(el, {
  animationClass: 'slideInUp'
});
// Available transitions
// fadeIn | slideInUp | slideInDown | zoomInDown | *
